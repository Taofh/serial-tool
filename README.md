# Serial Tool Pro

Web-based serial debugging tool powered by Web Serial API.

## Features

- Real-time serial communication via browser
- Support multiple baud rates, data bits, stop bits, parity
- HEX/ASCII display and send
- Auto scroll, timestamp, direction indicator
- Statistics (TX/RX bytes and frames)
- Search, export data
- Quick commands

## Usage

1. Open the page in Chrome/Edge 89+
2. Click "+" to add your serial device
3. Configure baud rate and other settings
4. Click "Connect" to start communication
